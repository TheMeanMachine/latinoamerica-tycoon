# Imágenes de Industrias

Este directorio debe contener las siguientes imágenes para las industrias:

1. `img/auto-factory.jpg` - Imagen de una fábrica de automóviles
2. `img/steel-plant.jpg` - Imagen de una planta siderúrgica
3. `img/textile-factory.jpg` - Imagen de una fábrica textil
4. `img/chemical-plant.jpg` - Imagen de una planta química

Las imágenes deben tener un tamaño recomendado de 400x300 píxeles y estar en formato JPG o PNG.

Puedes obtener imágenes de stock gratuitas de sitios como:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Pixabay (https://pixabay.com)

Asegúrate de que las imágenes sean libres de derechos de autor y adecuadas para uso comercial.
